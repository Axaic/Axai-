How to setup and start mining for AIMINING Token?

Just Click on the Mining.bat file and mining will be get started and Hashes will start stored on same folder with a file name “AIMINING.html”.
Open AIMINING.html to get the blocks information. Refresh the Browser page "AIMINING.html" after a certain time whenever you will find a new block.

If system alerts for a warning to run or download "Mining.bat", it's normal incase of Microsift windows. You can read the contents of "Mining.bat"
by right click on it and read "mining.bat" file in notepad. It must have below contents/commands:

start AIMINING.htm
java -jar AIMINING.jar g 1 AIMINING

How to stop or close mining application?

To Close the Mining process press CTRL+C in command prompt to get exit or close the Command prompt, but closing it properly with CTRL+C is good option.

Yes, it safe. Becuase even after sign the transaction it will ask for confirmation to release payment. After confirmation then payment will be released.

How much time it will take to mine a block?

Finding a block hash is totally random between 4294967296 hashes from nonce, depends on luck. Time to mine a block depends on CPU Speed. Faster the CPU, faster the probability to find block.
